# dotfiles

## Installation

1. Clone this repository
```
git clone --recursive https://github.com/hirotask/dotfiles.git
cd dotfiles
```

2. Install
```
./install.sh
```

3. Run the Wezterm
```
wezterm
```
```
```

4. (If you want to use zsh, run)
```
exec zsh
```

5. Enjoy!
